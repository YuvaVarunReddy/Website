    CSS Border-color

    The border-color property is used to set the color of the four borders.

    name - specify a color name, like "red"

    HEX - specify a HEX value, like "#ff0000"

    RGB - specify a RGB value, like "rgb(255,0,0)"

    HSL - specify a HSL value, like "hsl(0, 100%, 50%)"

    transparent

    Example:

    <!DOCTYPE html>

    <html>

    <head>

    <title>CSS Comments:</title>

    <style>

    p.one { border-style: solid; border-color: red; }

    p.two { border-style: solid; border-color: green; }

    p.three { border-style: dotted; border-color: blue; }

    p.four{ border-style: solid; border-color: red green blue yellow; }

    p.five{ border-style: solid; border-color: #ff0000; }

    p.six{ border-style: solid; border-color: #0000ff; }

    p.seven{ border-style: solid; border-color: #bbbbbb; }

    </style>

    </head>

    <body>

    <h2>The border-color Property</h2>
    <p>This property specifies the color of the four borders:</p>
    <p class="one">Some text.</p>
    <p class="two">Some text.</p>
    <p class="three">Some text.</p>
    <p><b>Note:</b> The "border-color" property does not work if it is used
    alone. Use the "border-style" property to set the borders first. </p>

    <h2>The border-color Property</h2>
    <p>The border-color property can have from one to four values (for the top border, right border, bottom border, and the left border):</p>
    <p class="one">A solid multicolor border</p>

    <h2>The border-color Property </h2>
    <p>This property specifies the color of the four borders:</p>
    <p class="five">A solid red border</p>
    <p class="six">A solid green border</p>
    <p class="seven">A dotted blue border</p>
    <p><b>Note:</b> The "border-color" property does not work if it is used alone. Use the
    "border-style" property to set the borders first.</p>

    </body>

    </html>
