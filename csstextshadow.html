    CSS Text-Shadow

    The text-shadow property adds shadow to text.

    Example:

    <!DOCTYPE html>

    <html>

    <head>

    <title>CSS Text-Shadow</title>

    <style>

    .one { text-shadow: 2px 2px; }

    .two { text-shadow: 2px 2px red; }

    .three { text-shadow: 2px 2px 5px red; }

    </style>

    </head>

    <body>

    <h1class="one"> Text-shadow effect!</h1>>

    <h1class="two"> Text-shadow effect!</h1>>

    <h1class="three"> Text-shadow effect!</h1>>

    </body>

    </html>

    Output:


    Text-shadow effect!
